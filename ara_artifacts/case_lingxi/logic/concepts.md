# Concepts

## Lingxi Adaptive Suite

The current production menu. It selects conservative fixed sleeves by scenario rather than using one global router.

## Sleeve

A prevalidated strategy component such as `lingxi`, `lingxi_pitnorm`, or `lingxi_pitnorm_gate_return_floor`.

## Conservative Promotion Gate

The rule that a candidate can enter production only after frozen out-of-sample evidence beats the relevant static menu under transaction costs.

## Validation Echo Trap

The pattern where a method wins validation Sharpe but fails after being frozen and tested out of sample.

## Agentic Strategy Evolution

Use of agents to propose, implement, critique, and document candidate strategies. Agents do not directly emit production trades.

## Structured Market Tags

Point-in-time market context labels that can be generated by deterministic rules or timestamped LLM debate. Tags are context features, not direct orders.
